## 🚀 About The Project

This is one of my early animation practices, focused on creating motion and visual interaction without using JavaScript.

The entire scene — including planetary orbits and movements — is built using:

* CSS `animation`
* CSS `transform`
* Basic HTML structure

The goal was to explore how far pure CSS can go in creating dynamic and engaging visuals.

---

## ✨ Features

* Smooth orbital animations
* Fully responsive layout
* No JavaScript — animation handled entirely with CSS
* Clean and minimal structure

---

## 🛠️ Built With

* HTML5
* CSS3 (Animations & Transforms)

---

## 📚 What I Learned

* Working with `transform` (rotate, translate, scale)
* Creating circular motion using keyframes
* Managing nested animations
* Improving visual timing and smoothness
